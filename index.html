Estagiário de Desenvolvimento Web

DECOM/CEFET-MG (Fev. 2022 - Mar. 2025)

Neste projeto, desenvolvemos uma plataforma web para correção de redações online, que será utilizada como uma ferramenta de ensino para os alunos e servidores do CEFET-MG.

Utilizamos o framework Django para criar o projeto, um servidor da Digital Ocean para hospedar, o Bitbucket para controle de versão e PostgreSQL para o banco de dados. Desenvolvemos as atividades de Front-end utilizando HTML, CSS e Bootstrap e Back-end utilizando Python, Django e JavaScript.

Durante o tempo de trabalho entregamos o servidor completo para que um grupo de alunos e servidores pudessem fazer testes na plataforma.

Estagiário de Machine Learning

DECOM/CEFET-MG (Mar. 2022 - Mar. 2025)

Este projeto é um projeto de pesquisa (Iniciação Cientifica), onde trabalhamos com a análise de dados de estruturas de concreto armado, com o objetivo de desenvolver uma ferramenta para prever a vida útil de estruturas de concreto armado.
